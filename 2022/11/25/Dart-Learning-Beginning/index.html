示例代码

dart

可以看出 Dart 的语法结构有点像 Java + Kotlin。有这两种语言基础的朋友应该不用注释都可以看懂。

主要概念

  • Dart 中所有可以被赋值给变量的都属于对象(Object)，包括 numbers, functions, null。
  • 虽然 Dart 是强类型语言，它可以省略类型的声明，编译器会进行类型推断。（跟 Kotlin 相似）
  • 类型后面加 ? 表示为可空变量，变量后面加 ! 表示断言这个变量一定不为空。（跟 Kotlin 相似）
